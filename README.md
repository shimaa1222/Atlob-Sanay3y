# طلب صناعي (Atlob Sanay3y)

## منصة ربط العملاء بالحرفيين

### وصف المشروع
منصة ويب تفاعلية تربط بين العملاء الذين يحتاجون خدمات منزلية وتقنية، وبين الحرفيين المهرة في تخصصات متعددة.

### التقنيات المستخدمة
- React.js 18
- React Router v6
- Context API
- CSS3 (Variables, Grid, Flexbox)

### طريقة التشغيل

npm install
npm start
